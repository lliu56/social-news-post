# Voice

This file defines the voice the skill uses when it writes posts. Edit it to match how you actually talk.

## Default voice — generic casual

Write like you're texting a friend who works in tech. Casual, direct, opinionated. Numbers and named companies do the heavy lifting. Reactions are short.

## Hook examples

The hook is the first line of the post. One line, lowercase, no period — or a quoted phrase from the headline. Some shapes that work:

- A short reaction word: `wow`, `huh`, `okay then`, `well damn`
- A quoted phrase from the headline in quotes
- A casual aside: `not sure how to feel about this`, `read this twice to make sure`
- A direct take: `feels like every week now`, `the numbers keep getting bigger`

Avoid:
- Marketing words (discover, unlock, transform, leverage, etc.)
- AI-cliche reframes ("It's not X, it's Y", "X isn't just Y, it's Z")
- Em dashes
- Hashtags in the hook

## Engagement question

The line right before the `--` separator. Has to invite an actual opinion, not a yes/no.

Good shapes:
- `what do you think this means for [X]?`
- `who do you think wins here?`
- `is this overhyped or undercounted?`
- `where does this leave [smaller player]?`

Bad shapes:
- `do you agree?` (yes/no)
- `thoughts?` (lazy)
- `what do you think?` (too generic)

## Tone notes

- One sentence per paragraph. Blank line between each.
- Numbers in every fact paragraph. No vague claims.
- Don't try to sound smart. Just say the thing.
- Short words beat long words.

## How to customize

If you have a specific catchphrase, opener, or sign-off you use, drop it in here. The skill reads this file before drafting any post, so whatever shape you write will become the default.
